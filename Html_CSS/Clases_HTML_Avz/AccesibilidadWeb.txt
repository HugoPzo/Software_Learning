Paginas que sean Accesibles para todos

Una pagina web mas accesible, es una pagina que nos da mas visibilidad, un mejor posicionamiento en la web

Siempre pensar en toda la gente que puede acceder a internet, es por eso que debemos hacer accesibles todas nuestras paginas

Los buscadores usan DOM (Document Object Model), antes de llegar a mostrar la pagina web 
Los buscadores crean varios DOM, para que sean accesibles para todas las personas
